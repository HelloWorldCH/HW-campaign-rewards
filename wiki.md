# คู่มือการใช้งาน PoE2 Campaign Rewards (สำหรับผู้ใช้ทั่วไป)

ยินดีต้อนรับสู่โปรแกรม **PoE2 Campaign Rewards**! โปรแกรมนี้คือ Overlay ที่จะช่วยบอกว่าด่านที่คุณกำลังเล่นอยู่ใน Path of Exile 2 มีของรางวัลอะไรบ้าง (เช่น Passive Points, Spirit, Stats) รวมถึงมีฟีเจอร์ Run Guide คอยบอกทางและสิ่งที่ต้องทำในแต่ละด่านแบบเรียลไทม์

## 📥 วิธีการดาวน์โหลดและติดตั้ง

1. เข้าไปที่หน้า [Releases ของโปรเจกต์](https://github.com/HelloWorldCH/HW-campaign-rewards/releases)
2. มองหาเวอร์ชันล่าสุด (Latest) และดาวน์โหลดไฟล์ `.exe` (เช่น `PoE2-Campaign-Rewards-Setup.exe` หรือ `PoE2-Campaign-Rewards-Portable.exe` ถ้ามี)
3. หากเป็นตัว Setup ให้ดับเบิลคลิกเพื่อติดตั้งตามปกติ หากเป็นตัว Portable สามารถดับเบิลคลิกเปิดใช้งานได้ทันที

---

## 🚀 การเริ่มต้นใช้งานครั้งแรก

เมื่อเปิดโปรแกรมขึ้นมาครั้งแรก สิ่งที่สำคัญที่สุดคือ **การตั้งค่าไฟล์ Client.txt** เพื่อให้โปรแกรมรู้ว่าตอนนี้ตัวละครของคุณอยู่ในด่านไหน

### วิธีตั้งค่า Client.txt
1. คลิกที่ปุ่มตั้งค่า ⚙️ (รูปเฟือง) ที่มุมขวาบนของโปรแกรม
2. ในหัวข้อ **Client.txt Path** ให้คลิกที่ปุ่ม `Select File`
3. ไปที่โฟลเดอร์เกม Path of Exile 2 ของคุณ (ปกติจะอยู่ที่ `C:\Program Files (x86)\Grinding Gear Games\Path of Exile 2\logs` หรือในโฟลเดอร์ Steam)
4. เลือกไฟล์ `Client.txt`
5. จากนั้นโปรแกรมจะเริ่มทำงานและสามารถตรวจจับการเปลี่ยนแผนที่ของคุณได้อัตโนมัติ

---

## 👤 ระบบโปรไฟล์ตัวละคร (Character Profiles)

หากคุณเล่นหลายตัวละคร (Alts) คุณคงไม่อยากให้ของรางวัลที่เคยรับไปแล้วของตัวละครแรก มาปะปนกับตัวละครใหม่

1. เปิดหน้าต่างตั้งค่า ⚙️
2. ในส่วนของ **Character Profiles** คุณสามารถพิมพ์ชื่อตัวละครใหม่ แล้วกด **Add Profile** ได้เลย
3. คุณสามารถเลือกสลับตัวละคร (Switch Profile) หรือลบตัวละครที่ไม่เล่นแล้ว (Delete) ได้ที่นี่
4. เมื่อคุณเปลี่ยนโปรไฟล์ สถานะการรับของรางวัลและการทำเควสในหน้า Run Guide จะถูกแยกออกจากกันอย่างสมบูรณ์!
5. ที่มุมซ้ายบนของจอหลัก จะมีจุดแสดงตัวละครปัจจุบันที่คุณกำลังใช้งานอยู่

## 🗺️ ระบบบอกทาง (Run Guide)

โปรแกรมมีฟีเจอร์สำหรับบอกเส้นทางการวิ่งแคมเปญให้เร็วที่สุด
1. คลิกที่ปุ่ม 🗺️ (รูปแผนที่) ที่มุมขวาบน เพื่อเปิดหน้าต่าง Run Guide
2. คุณจะเห็นเช็คลิสต์สิ่งที่คุณต้องทำในแต่ละด่าน
3. เมื่อคุณเข้าสู่ด่านใหม่ โปรแกรมจะเลื่อนหน้าต่าง Run Guide ไปยังด่านนั้นให้คุณโดยอัตโนมัติ (Auto-scroll)
4. เมื่อคุณทำภารกิจเสร็จ คุณสามารถติ๊กถูก ✅ ในเช็คลิสต์เพื่อบันทึกความคืบหน้าได้

---

## 📋 ระบบรางวัล (All Rewards)

อยากรู้ว่าเกมนี้แจกพอยท์ตรงไหนบ้าง?
1. คลิกที่ปุ่ม 📋 (รูปกระดาน) ที่มุมขวาบน เพื่อเปิดหน้าต่างแสดงรางวัลทั้งหมด
2. คุณจะเห็นรายชื่อเควสและด่านที่ให้ของรางวัลประเภทต่าง ๆ (Passive Skill, Spirit, และ Resistance)
3. สามารถดูได้เลยว่าคุณพลาดจุดไหนไปบ้าง

---

## ⌨️ คีย์ลัด (Hotkeys)

- โปรแกรมถูกออกแบบมาเป็น Overlay คุณสามารถกดปุ่มย่อ ─ เพื่อลดขนาดมันลง หรือกดปิด ✕ เพื่อซ่อนโปรแกรมได้
- หากซ่อนโปรแกรมไปแล้ว คุณสามารถเปิดกลับขึ้นมาใหม่ผ่าน System Tray (ไอคอนมุมล่างขวาของ Windows) ได้เสมอ
